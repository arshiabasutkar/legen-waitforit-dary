Core Ideas of Computer Science
  • function- abstraction of hard coding
  • ie. happybirthday.py print("Happy Birthday to you!").
  • From that point on when you want a sprite to do something, you just use the function instead of hard coding it everytime. A real life example would be moving your fingers. Your mind does not go through the hard coding in your brain to complete the action. It has become something you don't think about in that way like a function.
  • variable- an input that gives a patterned output
  • ie. counter = 100 # An integer assignment print counter.
  • A real life example would be an assigned number to a person in a classroom.
  • loop- a repeated action
  • ie. for x in range(0, 3): print "We're on time %d" % (x).
  • A real life example would be blinking that happens every so often.
  • conditional- an action contingent upon another action like an if then statement.
  • ie. if condition_1:statement_block_1 elif condition_2: statement_block_2 else:statement_block_3.
  • A real life example would be if you get a good night's sleep, then you will be energized the next day.
